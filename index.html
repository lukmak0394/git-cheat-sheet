My Git Cheat Sheet

Configuring user name and email

git config --global user.name 'Your Name'

git config --global user.email 'your@email.mail'

Opening folder in which we will keep project files

cd ~/directory/project-folder

Creating local repository in a specific folder

git init

Changing status to 'staged'

(git add *) OR (git add specific-file-name)

Commiting changes in local repository

git commit -m 'comment'

Connecting a local repository with a remote one

git remote add origin your-github-repo-path

Pushing a local repository to a remote one

git push -u origin branch-name (ex. main)
