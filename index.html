Basic Commands

Display current directory

pwd

Go to folder

cd folder_name

cd .. | folder up

cd ~ | home folder

Creates a folder

mkdir folder-name

Creates a file

touch file-name

Removing file with its' content

git rm -r file-name

Removing specific file

rm

Changing file name

mv old_filename new_filename

Copy file/folder to other folder

cp current_file-folder_path new_folder_path

Move file/folder to other folder

mv current_file-folder_path new_folder_path

Configuring GIT

Configuring user name and email

git config --global user.name 'Your Name'

git config --global user.email 'your@email.mail'

Working with repository

Creating local repository in a specific folder

git init

Changing status to 'staged'

(git add *) OR (git add specific-file-name)

Commiting changes in local repository

git commit -m 'comment'

Connecting a local repository with a remote one

git remote add origin your-github-repo-path

Pushing a local repository to a remote one

git push

Checking file status

git status

Repository clone

Cloning github pages repository

git clone https://github.com/username/username.github.io.git

Working with branches

Creating new branch

git checkout -b branch-name

Switching between branches

git checkout branch-name

Comparing branches

git diff branch1..branch2

Merging branches

git merge branch1 branch2
